# Code of Conduct

This project welcomes contributions from anyone. To keep collaboration
productive, we ask everyone interacting with the repository — issues,
pull requests, discussions, code review — to follow a few simple rules.

## Expected behaviour

- Be respectful and patient. Assume good faith.
- Critique ideas, not people.
- Focus discussion on the technical question at hand.
- Help others learn; nobody knows everything about this codebase.

## Unacceptable behaviour

- Personal attacks, insults, or demeaning comments.
- Targeted harassment of any kind.
- Off-topic content unrelated to the project.

## Reporting

If you observe behaviour that violates this code, please contact the
maintainer privately at **code@rebbert.de**. Reports will be handled
confidentially.

The maintainer reserves the right to remove comments, close issues or
pull requests, and block users whose conduct conflicts with this code.
